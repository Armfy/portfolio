Site interactif avec panel d'administration (AC24.02)

Modèle MVC & GitHub pour travailler en groupe (CE4.04)

J'ai utilisé Git et GitHub pour coordonner le développement en groupe avec mon binôme, gérer nos branches de fonctionnalités et fusionner notre code de manière sécurisée.

Interface 3D interactive connectée à une BDD (AC24.03)

J'ai conçu et programmé un site web intégrant un modèle 3D interactif et manipulable directement par l'utilisateur. La spécificité technique réside dans la liaison dynamique entre le modèle 3D et une base de données MySQL : chaque clic sur une zone spécifique du modèle interroge la base pour afficher les informations associées.

Intégrer des interactions riches pour le web (CE4.03)

J'ai géré l'ensemble de la chaîne technique : de l'optimisation des fichiers 3D pour le web à la gestion des événements de clic (raycasting), pour concevoir une expérience fluide sans ralentir le chargement de la page.

Mini-jeu de bowling interactif sous Unity (AC14.03)

J'ai développé un jeu de bowling sous Unity. J'ai programmé en C# les mécaniques physiques (force du lancer, trajectoire, collisions des quilles) et l'algorithme de calcul des points pour créer une boucle d'interaction simple et efficace.

Synthèse de mes apprentissages (C4)
